Welcome to the Movie API Documentation

This API provides access to a database of movies. Developers can use this API to search for movies, retrieve details about specific movies, and perform other related tasks.

Endpoints

Request URL Method Query Parameters Description Request Body Format Response Format
GET /movies GET N/A Retrieve a list of all movies N/A JSON array of movie objects
GET /movies/:id GET Movie ID Retrieve details about a specific movie N/A JSON object representing the movie
POST /movies POST N/A Add a new movie to the database JSON object with movie data JSON object representing the newly added movie
PUT /movies/:id PUT Movie ID in URL Update details of a specific movie JSON object with updated movie data JSON object representing the updated movie
DELETE /movies/:id DELETE Movie ID Delete a specific movie from the database N/A Success message indicating deletion
PUT /users/:id PUT User ID in URL Update user info JSON object with data to be updated:
